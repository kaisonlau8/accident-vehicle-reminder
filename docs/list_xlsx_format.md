# list.xlsx 格式说明

此文件为门店联系人主表，包含真实手机号，**不应提交到 GitHub**。

首次使用时，按以下格式创建 `list.xlsx`（放在项目根目录）：

## Excel 格式（14 列）

| 列号 | 列名 | 说明 | 示例 |
|------|------|------|------|
| A | 序号 | 行号 | 1 |
| B | 门店编码 | DMS 门店代码 | 510001 |
| C | 区域 | 区域名称 | 华中 |
| D | 门店名称 | 门店全名 | 武汉猛士授权服务中心 |
| E | 督导姓名 | 区域督导 | 张三 |
| F | 督导飞书电话号 | 督导手机号 | 13800138000 |
| G | 提醒人1 | 门店提醒人1姓名 | 李四 |
| H | 电话1 | 提醒人1手机号 | 13900139000 |
| I | 提醒人2 | 门店提醒人2姓名 | 王五 |
| J | 电话2 | 提醒人2手机号 | 13700137000 |
| K | 提醒人3 | 门店提醒人3姓名 | |
| L | 电话3 | 提醒人3手机号 | |
| M | 提醒人4 | 门店提醒人4姓名 | |
| N | 电话4 | 提醒人4手机号 | |

## 注意事项

- 提醒人3/4和电话3/4可以为空
- 督导飞书电话号必须是纯数字（不带+86前缀）
- 电话1~4也必须是纯数字
- 每次更新此表后，系统下次运行会自动读取最新数据
- Web 控制台的"门店联系人"页面也支持上传替换此文件
- 全国收件人不在 `list.xlsx` 中维护，请在 Web 控制台的"门店联系人"页面新增或删除
